# Proyecto WebF1 -- Grupo 1

Este repositorio contiene un sitio web completo relacionado con la
Fórmula 1, incluyendo páginas informativas, formularios, interactividad
con JavaScript y procesamiento básico con PHP.

##  Estructura del proyecto

### Raíz del proyecto

-   **index.html** -- Página principal del sitio.
-   **index.js** -- Lógica JavaScript para la página principal.
-   **acceso.html / acceso.js** -- Página y script para el acceso/login.
-   **comprar_entrada.html** -- Página para comprar entradas.
-   **entrada.js** -- Lógica asociada a la compra de entradas.
-   **contacto.html** -- Página con formulario de contacto.
-   **sorteo.html / sorteo.js** -- Página con información y
    funcionalidad del sorteo.
-   **procesar_compra.php** -- Procesa los datos enviados desde la
    compra de entradas.
-   **tratarDatos.php** -- Procesa datos varios, posiblemente desde
    formularios.

###  /css

Contiene las hojas de estilo del sitio. - **acceso.css**, **index.css**,
**entradas.css**, **contacto.css**, **sorteo.css**, entre otras.
Organizan estilos para cada una de las páginas principales.

###  /images

Incluye imágenes para: - Carreras - Logos de patrocinadores - Pilotos
(para el sorteo) - Iconos (como mostrar/ocultar contraseña) -
Fotografías varias usadas en el diseño

###  /images/sorteo

Imágenes de pilotos de F1 utilizadas en la página del sorteo.

###  /images/logos

Logos de patrocinadores como Rolex, Qatar, Salesforce, etc.

###  /adicionales

Material adicional de versiones previas u otras prácticas: -
**primera.html / primera.css / primera.js** - **procesar_compra.php**

##  Tecnologías utilizadas

-   **HTML5** -- Estructura del sitio.
-   **CSS3** -- Estilos y diseño responsive.
-   **JavaScript** -- Validaciones, interactividad y dinámica del sitio.
-   **PHP** -- Procesamiento básico de formularios.

##  Objetivo del proyecto

Ofrecer una experiencia web completa sobre Fórmula 1 que incluya: -
Información general. - Compra simulada de entradas. - Sorteos
interactivos. - Formularios de contacto y acceso.
